Historia del HTML

1991
HTML 1.0

Primera versión, muy básica. Solo permitía texto sencillo y enlaces.

1995
HTML 2.0

Se reconocen formularios y se estandarizan las etiquetas básicas.

1997
HTML 3.2

Se agregan tablas, estilos inline y soporte para scripts.

1999
HTML 4.01

Incluye separación entre estructura y estilo. Introduce CSS de forma oficial.

2000
XHTML

Versión más estricta basada en XML.

2014
HTML5

Versión moderna: multimedia nativa, semántica, APIs, formularios mejorados, etc.
